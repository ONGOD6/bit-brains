# Bit Brains — Ethereum Improvement Proposals

This repository contains the official technical specifications for the **Bit Brains ecosystem**.

Bit Brains is an Ethereum NFT ecosystem composed of two primary collections:

• Pickle Punks  
• Bit Brains

These specifications define the core standards governing the collections, including:

- supply limits
- metadata structure
- minting rules
- collection architecture

The goal of these documents is to provide transparent and simple standards for the ecosystem.
